# Landing Page Design Assignment - XIBOTIX

A modern, responsive deep-tech landing page built to present the two core operational sectors of XIBOTIX **Humanoid Robotics** and **Rehabilitation Devices** with equally balanced visual hierarchy.

## 🚀 Live Preview
* **Live Deployment:** [https://xibotix-robotics.netlify.app/](https://xibotix-robotics.netlify.app/)

## 🛠️ Technology Stack
* **Framework:** React.js (Vite)
* **Styling:** Tailwind CSS
* **Icons:** React Icons & Lucide React Icons
* **Deployment:** Netlify

## 📱 Key Features
* **Balanced Design:** Split layout ensuring neither division dominates the user experience.
* **Fully Responsive:** Fluid grid transitions optimized for standard mobile viewports up to wide desktop monitors.
* **Deep-Tech UI/UX:** High-contrast layout using clean topography tailored for engineering fields.
